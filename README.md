## 项目介绍

**一个基于SpringBoo + Vue + Element UI 开发的权限管理系统**

- 角色权限精确到菜单、接口访问
- 前后端通过jwt进行认证状态管理
- 持续更新...

## 技术栈

- 后端：SpringBoot、SpringSecurity+jwt、MyBatisPlus
- 前端：Vue-cli、Vuex、ElementUI
